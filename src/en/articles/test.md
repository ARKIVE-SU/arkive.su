We are thrilled to announce that articles can now be written in convenient Markdown format using the power of the Eleventy Static Site Generator!

## Why is this necessary?
Because pasting long texts into JSON files was a **complete nightmare**. Now you simply drop the ChatGPT-translated text into the `/en/articles/` folder, and it automatically appears on the site with perfect SEO and localized URLs.

* Fast publishing
* Perfect for AI-driven localization workflows
* Zero latency: the site is fully pre-rendered to HTML for maximum speed.
